# Проект: Виджет банковских операций

## Описание
Проект предназначен для подготовки данных для нового виджета личного кабинета клиента банка. Он включает функции маскировки номеров карт/счетов и инструменты для фильтрации и сортировки транзакций.

## Установка
1. Клонируйте репозиторий.
2. Установите зависимости через Poetry:
   ```bash
   poetry install
## Тестирование

Проект покрыт unit-тестами с использованием библиотеки `pytest`. 
Для запуска тестов и проверки покрытия используйте команду:

```bash
pytest --cov=src --cov-report=term-missing
